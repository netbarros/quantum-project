## 2.3 `fluxo-comunicacao-entre-agentes.png`
O diagrama mostra:
- Express API dispara `dispatchDailySession()`
- AgentRegistry chama:
  - MonetizationAgent: `execute(checkAccess)`
  - ContentAgent: `execute(getUserContext)` / `execute(generateContent)`
  - PersonalizationAgent consulta comportamento
  - AI Gateway gera
  - PostgreSQL persiste conteúdo e usage
  - ProgressAgent atualiza score/streak/level ao completar

### Decisão aplicada no PR
O PR define uma **orquestração síncrona por request**, com mensagens rastreáveis via:
- `id`
- `type`
- `userId`
- `correlationId`
- `timestamp`
- `metadata`

Isso permite:
- tracing por cadeia
- métricas por agente
- debug real de produção
- analytics por latência/modelo/custo

---

## 2.4 `MonetizationAgent-EnforcementFlow.png`
Fluxo explícito:
- Request chega
- usuário autenticado?
  - não → `401 Unauthorized`
- é premium?
  - sim → segue
- não premium:
  - `currentDay > 7`?
    - sim → paywall / upgrade required
  - não:
    - AI calls hoje > 3?
      - sim → warning + acesso liberado ou limite diário atingido
      - não → acesso liberado

### Decisão aplicada no PR
O PR implementa **dupla camada de monetização**:
1. **Hard gate de jornada**
   - free: até dia 7
2. **AI usage governance**
   - free: 3 chamadas/dia
   - premium: 50/dia soft limit

E distingue:
- `ACCESS_GRANTED`
- `ACCESS_GRANTED_WITH_WARNING`
- `ACCESS_DENIED_PAYWALL`
- `ACCESS_DENIED_RATE_LIMIT`

# 23. DEFINIÇÃO DE PRD EXECUTIVO

## Nome do produto
Quantum Project

## Versão
V2.0 System 6 — Antigravity Edition

## One-liner
Uma plataforma de transformação comportamental guiada por IA que converte prática diária em mudança de identidade.

## Problema
Apps de mindfulness e journaling falham em personalização, progressão e retenção emocional. O usuário até começa, mas não sente evolução viva.

## Solução
Um sistema ritualizado, adaptativo e progressivo, com conteúdo diário gerado a partir do contexto do usuário, reforçado por visualização de progresso, notificações inteligentes e limites de monetização que empurram upgrade apenas após prova de valor.
